# CycleHub - Bicycle Inventory & Sales Management System 🚲

CycleHub is a comprehensive management solution designed for bicycle shops to streamline their inventory tracking, sales processing, and customer financial management. This project was developed using Oracle APEX and SQL.

### 🚀 Features
* **Interactive Dashboard:** Real-time visualization of total inventory, sales, and stock status using dynamic charts.
* **Inventory Management:** Advanced search and filtering options to manage bicycle stock by brand, category, and condition.
* **Sales & Payments:** Track customer purchases, installments, and outstanding balances efficiently.
* **Automated Alerts:** Built-in notifications for Low Stock items and Overdue Customer Payments.
* **Responsive UI:** Clean and user-friendly interface optimized for business operations.

### 🛠️ Tech Stack
* **Platform:** Oracle APEX (Application Express)
* **Database:** Oracle SQL / PL/SQL
* **Styling:** CSS & Universal Theme

### 📥 How to Install
1. Download the `.sql` export file from this repository.
2. Log in to your **Oracle APEX Workspace**.
3. Go to **App Builder** > **Import**.
4. Upload the `.sql` file and follow the installation wizard.
5. Ensure **"Install Supporting Objects"** is selected to include the database tables and sample data.
